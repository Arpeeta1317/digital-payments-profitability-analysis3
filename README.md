# Digital Payments Profitability Analysis

## Business Problem
Despite increasing transaction volume, the company experienced declining profitability.
This project analyzes revenue, costs, refunds, retention, and marketing ROI.

## Datasets
- Transactions
- Users
- Merchants
- Costs
- Refunds
- Marketing

## Analysis
- Data cleaning & validation
- Revenue vs cost analysis
- Transaction funnel & failure analysis
- Cohort retention analysis
- Marketing ROI

## Tools
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab
